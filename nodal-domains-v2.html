Domains

ⓘ How to play

Color every cell blue or red so the grid has exactly N connected color regions (domains). Each puzzle tells you the target N.

👆Tap any cell to cycle its color: uncolored → blue → red → uncolored.
🔗Two cells belong to the same domain when they share an edge and have the same color. The small number in each cell shows its domain ID.
🔒Locked seed cells are fixed — they show you where the puzzle begins.
💡Stuck? Use the Hint button to reveal one correct cell.
